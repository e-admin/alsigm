Manual de usuario Estructura Organizativa con LDAP

 

El objetivo de esta aplicación es mostrar el organigrama de su empresa. Permite usar el Servicio de directorios utilizado por empresas u organizaciones para la identificación de usuarios.

Ventana de Administrador de usuarios

La pantalla principal está formada por los siguientes elementos básicos:

  • En la zona izquierda se muestra el árbol de usuarios o de grupos dependiendo del elemento elegido en la zona inferior.
  • En la zona inferior se muestra el cuadro de búsqueda, se podrá buscar por usuario o por grupo.
  • En la zona derecha, se muestran las propiedades del elemento elegido.

La estructura de usuarios del Servicio de directorios de la empresa u organización. Donde:

  • Representa el directorio que se ha indicado como Raíz.
  • Es el indicador genérico en el Servicio de Directorios.
  • Es el indicador de la estructura organizativa en el Servicio de Directorios.
  • Es el indicador de grupo en el Servicio de Directorios.
  • Es el indicador de usuario en el Servicio de Directorios.

Los usuarios con permiso para ello pueden realizar las siguientes operaciones:

  • Listar usuarios.
  • Listar grupos.
  • Buscar y consultar usuarios.
  • Buscar y consultar grupos.
Consultar las propiedades de Grupos y Usuarios

Cambiando el combo de búsqueda seleccionamos el tipo de datos que se van a mostrar en el árbol.

Cuando buscamos o seleccionamos un usuario, se abre una pantalla que muestra las propiedades:

Estas propiedades pueden variar según el usuario.

No se puede modificar ningún dato en esta pantalla.

Buscar grupos y usuarios

Es probable que el sistema cuente con un gran número de usuarios asociados a grupos y departamentos. A menudo es útil que el administrador del sistema pueda consultar los perfiles de dichos usuarios sin necesidad de recordar el departamento o los grupos a los que éste está asociado. Lo mismo ocurre con los propios grupos y departamentos definidos.

Para facilitar esta operación, se ha diseñado la opción Buscar disponible la barra de búsqueda. Seleccionando el criterio de búsqueda, Grupo o Usuario, y escribiendo el nombre completo del elemento que esté buscando, el sistema le situará directamente donde se encuentre dicho elemento.
